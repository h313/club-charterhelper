About

This was a thing I made some time ago to help with clubs at my school. It allows for people to easily sign up for or drop out of clubs and allows all users to make a club. All those subscribed to the club will have access to a list of all members in the club. To log in, you will need a Google account with the correct email domain (in this case, charterschool.org).

Usage

Using this is easy, check out the GitHub repository for the source code (minus my API key) and usage instructions.
